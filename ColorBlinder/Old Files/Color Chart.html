Main Digest

Chart of HTML color samples including Hex color codes, RGB values, and website safe accessibility colors for people with Protanopia, Deuteranopia, or Tritanopia.

Good graphic design avoids using color coding or color contrasts alone to express information, as this not only helps people who are color blind, but also aids understanding by normally sighted people.

The use of Cascading Style Sheets (CSS) on the Internet allows web pages to be given an alternative color scheme for color-blind readers. You may also find our color chart handy to help find ideal co-ordinated color schemes and palettes that complement one another for website themes and designs.

Web Page Design Tips for Color blind Users

Color codes present particular problems for color blind people as they are often difficult or impossible for color blind people to understand.

  • Red and green can be hard for a color blind person to tell apart. Avoid placing red and green colors together.
  • Don't use only color to highlight something specific on your web page.
  • De-saturate web page images to see if they still have impact.

Terminology


Screenshot of the website color reference chart shown below.

Protanopia, deuteranopia, protanomaly, and deuteranomaly are commonly inherited forms of red-green color blindness which affect a substantial portion of the human population. Those affected have difficulty with discriminating red and green hues due to the absence or mutation of the red or green retinal photoreceptors. The term red-green color blindness accurately describes this most common form of color vision deficiency.

Trichromacy means having normal color vision which uses all 3 types of light cones correctly, people with normal color vision are known as trichromats.

Achromatopsia (ACHM) means total color blindness and is medically defined as the (possible) inability to see any color.

  • RGB Values: Usually given in the 0 to 255 range; if they are in the 0 to 1 range, the values are then multiplied by 255 before conversion.
  • Hex triplet: A hex triplet is a six-digit, three-byte hexadecimal number used in HTML, CSS, SVG, and other computing applications to represent colors.
